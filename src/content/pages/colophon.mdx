A short note on how this site is made — partly for the curious, partly because we believe the medium is part of the message.

## Typography

Headlines are set in **Fraunces**, a display serif whose optical-size axis is pushed to its display setting for the large editorial moments. Everything else — body copy, labels, and technical specs — is **Geist** and **Geist Mono**. All three are self-hosted, so the site renders correctly offline and makes no third-party font requests.

## Built with

- **Astro** — for the static, content-first architecture and View Transitions.
- **Tailwind CSS** — with a small, role-named token system, so changing a handful of variables re-skins the whole site.
- **Alpine.js** — for the few interactive touches (the menu, the theme toggle) that genuinely need state.

## Principles

- Performance is a feature: self-hosted fonts, responsive images, and no tracking.
- Light and dark are first-class, not an afterthought.
- The content lives in plain data and Markdown files, so the words can change without touching the layout.
